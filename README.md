## 中国 AI 开发者项目列表
聚合所有中国 AI 开发者的项目                  


**1. 为什么有这个表**    
作为开发者其实比较好奇其他人在做什么业余项目（不管目的是做到盈利/玩票/试试看）               
所以特意建了这个库。欢迎各位开发者把自己的项目加进来~ 发 Pull Request 或 Issue 即可 <br/>
（入选标准：必须是网站或App，不能是开发者工具或论坛型网站）   

**2. 项目有 3 种状态**          

| 开发中 | 已上线 | 已关闭 |
|--------|--------|--------|
| :clock8: | :white_check_mark: | :x: |
